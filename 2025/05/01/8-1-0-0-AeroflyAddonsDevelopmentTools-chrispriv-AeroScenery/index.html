This is an updated MOD for @nickhod ‘s popular App for creating photo sceneries on Aerofly FS2/4, features many improvements comparing to the initial version (e.g. works together with my TreesDetection App, allows to create worldwide 30m elevation maps in a simple way for AFS, to download OSM data separately, new with simultaneous downloads up to 8 streams instead of 4, makes several satellite image sources to work again and much more, enhanced seach and much more). Now also usable as a moving map for Aerofly.

Highlights of updated version 1.1.3 MOD i & j:

※Now also usable as a moving map for Aerofly reading UDP data port with data streaming ‘on’ in Aerofly FS2/4 Settings.
※Added “map fixed”, “flight tracing” & “hide working tiles” mode, detecting IP address functionality and interpolation for smother movement in moving map.
※Enhanced Search Tile/Location function, which also allows you to search for places, regions etc. (performs geocoding for location search using OpenStreet Map Data).
※Added tooltips and hints making the use of AeroScenery easier for beginners (working scenery name can be set now under Settings for easy installation using “Install Scenery” for selected tiles after conversion).
※Simpler installation of the MOD without prior installation of the update to version 1.1.3 is now possible, since the whole package is included.

Installation of MOD

  1. You first need Geoconvert tool as part of Aerofly FS 2 SDK (not supported by IPACS anymore but still available for download here: Aerofly FS 2 Software Development Kit (SDK) – Aerofly-Sim.de )

2.Install the initial Release 1.0.1 of AeroScenery from GitHub (GitHub - nickhod/aeroscenery > Microsoft Windows Installer AeroScenery.msi ). Make sure you have a correct and properly working initial Release 1.0.1 of AeroScenery, including working settings *)

  1. Just overwrite the whole “C:\Program Files (x86)\AeroScenery” folder with the attached and unzipped MOD 1.1.3 folder (check the file specs of the executable file and the version number of the MOD in the upper right corner after the first launch of the modified AeroScenery App)

*) After the installation using the installer, you should have a new directory in your user directory “\Documents\AeroScenery" containing the setting.xml and the subfolders “database” and “working”. Finally, you have to set under Settings “AFS2 SDK Folder” of AeroScenery App the path to the root folder of IPACS Aerofly FS2 SDK (aerofly_fs_2_geoconvert.exe should then be located by default in the subdirectory “\aerofly_fs_2_geconvert").

If you still have issues, there are several workarounds:

a) Increase the values in Settings up to “30 / 10”

b) Redownload the tiles and then stitch the image tiles again (using the MOD, now only missing tiles would be downloaded)

c) The fastest and most reliable way to download missing tiles is to run the provided PowerShell PS1 script and stitch the Image tiles again
